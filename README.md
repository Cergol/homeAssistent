# homeAssistent
Мой проект для познания
Данный проект необходим для улучшения навыков программирования и изучения новых технологий и методов

Начну с БД. Хочу в БД хранить заметки, чтобы не забывать вещи. Например какой серик начал смотерть, что хотел сделать/купить и т.д. Соответственно нужна функция для добавления в БД, удаление из БД. Должен быть отдельный заголовок, дата создания, описание. Может быть добавить возможность подзадач. Но как задать эти подзадачи. Самое простое это json. у него будет подзадача и статус. Тогда нужно будет уметь обновлять его.
